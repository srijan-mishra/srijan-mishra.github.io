Google Summer of Code 2015

Google Summer of Code is a global program. Students work with an open source organisation on a 3 month programming project.

I got an exciting opportunity through GSOC '15 to work closely with the organization Pidgin, Finch and libpurple under the guidance of my mentor Ethan Reisner

libpurple is an open source IM (instant messaging) library built using C. It powers various IM clients like Pidgin, Finch and Adium.

Pidgin is an open source GTK+ based chat client used by millions, built on top of libpurple. Pidgin supports various protocols viz. GTalk, XMPP, IRC, AIM, Yahoo etc.

Motivation

Pidgin currently supports GTalk through the open XMPP protocol. XMPP protocol is no longer supported by Google in favour of the newer proprietary Google+ Hangouts protocol. Apart from this the GTalk XMPP does not support group chat, audio and video calls.

A native implementation of the newer Google+ Hangouts protocol in libpurple will ensure continued support of the widely used IM services in libpurple based IM clients.
